About

This website was made by a secondary school teacher who values learning through fun! Sometimes in class there are a few minutes near the end where students start to disengage. The activities on this site aim to fill that gap and maximise effective learning time.

If you want to support my work and help keep this resource free for everyone, a small donation goes a long way.

Tips & Tricks

Many of these activities are flexible in how you run them. They can be whole class activities or individual work. These activities don't necessarily need this website to run and can often be done with a whiteboard.
